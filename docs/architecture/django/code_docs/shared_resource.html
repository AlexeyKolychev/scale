class shared_resource.models.SharedResourceRequirement(*args, **kwargs)¶

Bases: django.db.models.base.Model

A requirement for a shared resource. A job is an example of something that can have a global resource requirement. The requirement may include a usage amount. As an example you could require 1.3 Gbps from a shared network pipe.
