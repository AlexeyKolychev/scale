create_batch(*args, **kwargs)¶

Creates a new batch that represents a group of recipes that should be scheduled for re-processing. This method also queues a new system job that will process the batch request. All database changes occur in an atomic transaction.

Parameters:
  • recipe_type (recipe.models.RecipeType) – The type of recipes that should be re-processed
  • definition (batch.configuration.definition.batch_definition.BatchDefinition) – The definition for running a batch
  • title (string) – The human-readable name of the batch
  • description (string) – An optional description of the batch
Returns:

The newly created batch
