get_error(exit_code=None, default_error_name=u'algorithm-unknown')¶

This method retrieves the error that maps to the given exit code. If the exit code is zero (success) None is returned. For a failure exit code with no mapping, the given default error is returned.

Parameters:
  • exit_code (int) – The exit code from a task
  • default_error_name (string) – The name of the error to use if no mapping exists for the exit code.

fully_populate_command_argument(job_data, job_environment, job_exe_id)¶

Return a fully populated command arguments string. If pre-steps are necessary (see are_pre_steps_needed), they should be run before this. populated with information from the job_data, job_environment, job_input_dir, and job_output_dir.This gets the properties and input_files from the job_data, the shared_resources from the job_environment, and ${input} ${output_dir} from the work_dir. Throws a job.configuration.interface.exceptions.InvalidEnvironment if the necessary pre-steps have not been performed
